## GTS Fiber (Static HTML/CSS/JS)

This is a fully static website. No PHP and no backend required.

### Pages
- `index.html` (Home)
- `products.html`
- `about.html`
- `contact.html`

### Assets
- `assets/css/style.css`
- `assets/js/main.js`

### Media
- **Hero image**: `images/hero.jpg`
- **Home video (EN)**: `videos/gts-english.webm` (no autoplay; click play)
- **Home video (AR)**: `videos/gts-arabic.webm` (no autoplay; click play)

### Run
- Open `index.html` in your browser (double-click).
